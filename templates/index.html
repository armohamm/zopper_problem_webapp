Zopper Interview Problem


Abstract:

This application has been developed as a part of Zopper Interview for developing an application.
Application reads the pincode directory of India from a csv file downloaded from Pincode Directory.

Click here to go to the online view of directory.

This IRV system just wont only help NGO to automate donations but also different NGOs can share a single number for the same purpose.

Using Search API

Search API Base Request URL:- http://zopper.herokuapp.com/searchAPI
pass the parameter as list of tuples.
For example:
data = [(criteria_no,'filter criteria')]
Below parameter has only one criteria which looks for pincode containing 800002

data = [('field0', u'pincode,contains,800002')]
Below parameter has two criteria which looks for taluk beginning with 'Banga' or statename is 'assam'

data = [('field0', u'Taluk,begins,Banga'),('field1','statename,is,assam')]
A maximum of 4 criterias can be used to filter the result.
Pincode, Taluk, District Name, Statename along with is,contains,begins or ends condition.
res = requests.post('http://zopper.herokuapp.com/searchAPI',params=data)
Response will be in json format
